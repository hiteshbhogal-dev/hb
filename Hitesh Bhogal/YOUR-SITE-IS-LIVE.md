## 🔍 **If You're Seeing Old Version:**

Sometimes browsers cache the old version. Try these steps:

### **On Mobile:**

**iPhone (Safari):**
1. Tap the **aA** icon in the address bar
2. Select **"Reload Without Content Blockers"**
3. Or: Settings → Safari → Clear History and Website Data

**Android (Chrome):**
1. Tap the **⋮** menu (3 dots)
2. Select **"Settings"**
3. Go to **"Privacy"** → **"Clear browsing data"**
4. Check **"Cached images and files"**
5. Tap **"Clear data"**
6. Reload your site

### **On Desktop:**

**Hard Refresh:**
- **Windows:** Ctrl + Shift + R (or Ctrl + F5)
- **Mac:** Cmd + Shift + R
- **Linux:** Ctrl + Shift + R

**Or Clear Cache:**
1. Open DevTools (F12)
2. Right-click the **refresh button**
3. Select **"Empty Cache and Hard Reload"**

## 🐛 **Troubleshooting:**

### **"I don't see the hamburger menu"**
- Solution: You might be on desktop view
- The hamburger only shows on screens ≤ 768px wide
- Try opening on your phone or resize browser window

### **"The menu doesn't open"**
- Solution: Hard refresh the page (clear cache)
- Try: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

### **"I don't see the favicon"**
- Solution: Clear browser cache
- Or: Close and reopen the tab
- Favicon might take a minute to load

### **"Layout still looks broken on mobile"**
- Solution: Hard refresh (clear cache)
- The styles are already deployed
