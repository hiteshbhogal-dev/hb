## 🚀 Setup Instructions

### For the Chatbot (Required):

**Step 1:** Get FREE Google Gemini API Key
1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with Google account
3. Click "Create API Key"
4. Copy the key

**Step 2:** Add API Key to Code
1. Open `js/chatbot.js`
2. Find line 5
3. Replace with your API key:
```javascript
const GEMINI_API_KEY = 'YOUR_API_KEY_HERE';
```

**Step 3:** Test
1. Open `index.html`
2. Click orange chat button
3. Ask: "Tell me about Hitesh"
