```
┌──────────────────────┐

```

### **Open State:**
```
┌──────────────────────┬───────────┐
│ HB               ✕   │  Home     │
│                      │  About    │
│   [Main Content]     │  Experience

│                      │  Awards   │
│                      │  Insights │
│                      │  Contact  │

```

### **Features:**
- Tap hamburger → menu slides in from right
- Tap X → menu slides out
- Tap any link → navigate + close menu
- Tap outside → close menu
- Smooth animations (0.4s)
- No background scroll when open
- Touch-friendly 44px+ tap targets
